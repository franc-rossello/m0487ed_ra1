RA11 Reconeix la relació dels programes amb els components del sistema informàtic: memòria, processador, perifèrics, entre d'altres.


Com interactuen els programes amb els elements de maquinari per funcionar?

Els programes informàtics necessiten interactuar de manera constant amb els components del sistema informàtic per poder funcionar correctament. A continuació, es descriuen els components clau d'un sistema informàtic i com es relacionen amb els programes per garantir el seu funcionament òptim.

Processador

El seu paper principal és executar les instruccions dels programes, és a dir, realitzar els càlculs i operacions lògiques que els programes li indiquen.

Memòria

La memòria és essencial perquè un programa pugui funcionar. Quan s'executa un programa, aquest necessita un espai on guardar dades temporals i instruccions.

Perifèrics

Els programes necessiten aquests components per interactuar amb l'usuari i amb el món exterior. Els Perifèrics són dispositius externs que s'utilitzen per interactuar amb el sistema informàtic.
Es poden classificar en dispositius d’entrada, sortida o emmagatzematge.

Sistema Operativo

El SO s’encarrega de gestionar la memòria, controlar els dispositius d’entrada i sortida i gestionar els processos. El SO és el programari que gestiona tots els components de l'ordinador.
Facilita la interacció entre els programes i el maquinari.


Funcionament Processador

Cicle d'Execució

  • Fetch (obtenir): El processador carrega una instrucció des de la memòria.
  • Decode (decodificar): El processador interpreta la instrucció per entendre quin tipus d'operació ha de realitzar..
  • Execute (executar):El processador porta a terme l'operació que s'indica en la instrucció (per exemple, una operació aritmètica, lògica, etc.).

Paral·lelisme

Els processadors moderns poden executar múltiples instruccions alhora gràcies al paral·lelisme. Això es fa possible a través de:

  • Múltiples Nuclis: Diverses unitats de processament dins d'un mateix xip que poden executar tasques simultàniament.
  • Pipelining: Tècnica on es preparen diverses instruccions mentre es realitza una altra, com una cadena de muntatge.
  • Hyper-Threading: Permet a cada nucli físic processar diversos fils d'execució simultàniament.

Tipus de Memòria

RAM Random Access Memory

  • Memòria de lectura i escriptura.
  • Emmagatzema dades temporals.
  • Esborra quan s'apaga l'ordinador.
  • Accés ràpid pel processador.

ROM Read-Only Memory

  • Emmagatzema dades permanents.
  • No es modifica (només lectura).
  • Conserva dades sense energia.
  • Conté firmware bàsic.

Memòria Flash

  • Conserva dades sense energia.
  • Utilitzada en USB, SSD, targetes.
  • Més ràpida que discs durs.
  • Resistent als cops.

Memòria Virtual.

  • Utilitza espai del disc del sistema o memòries flash.
  • Permet executar més programes.
  • Més lent que la RAM física.
  • Gestió pel sistema operatiu.

Tipus de Perifèrics

Perifèrics d'Entrada: Permeten introduir dades o ordres.

  • Teclat
  • Ratolí
  • Pantalla tàctil
  • Escàner

Perifèrics de Sortida: Mostren o envien dades processades.

  • Monitor
  • Impressora
  • Altaveus
  • Projector

Emmagatzematge: Guarden dades permanentment.

  • Disc dur (HDD)
  • Unitat SSD
  • Discs òptics
  • Memòries USB

Comunicació: Transferència de dades

  • Adaptador Wi-Fi
  • Ports USB
  • Bluetooth
  • Xarxa cablejada
